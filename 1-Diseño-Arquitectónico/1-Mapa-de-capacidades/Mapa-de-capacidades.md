## Módulo 1: Planificación  
Este módulo se centra en el diseño estratégico del servicio y la optimización de recursos antes de la operación diaria, buscando maximizar la eficiencia y satisfacer la demanda.
![Mapa Capacidades](https://files.catbox.moe/ep5u6h.jpg)

### 1.1 Diseño de rutas y frecuencias

| Nivel 3 (Función)               | Nivel 4 (Sub-Función)                  | Nivel 5 (Detalle)                                      | Explicación                                                                 |
|---------------------------------|----------------------------------------|--------------------------------------------------------|------------------------------------------------------------------------------|
| Análisis de datos de demanda    | Recopilación de datos externos         | Obtención de información de fuentes externas           | Evalúa patrones de movilidad para ajustar rutas y frecuencias.              |
|                                 | Analítica de datos                     | Número de rutas                                        | Determina cuántas rutas son necesarias.                                     |
|                                 |                                        | Cantidad de pasajeros transportados                    | Mide la demanda real del sistema.                                           |
|                                 |                                        | Origen - destino                                       | Identifica los principales flujos de viaje.                                 |
| Optimización de rutas           | Simulación de rutas alternativas       | Modela escenarios para comparar opciones               | Busca mejorar eficiencia y tiempos de viaje.                                |
|                                 | Evaluación de tiempo de viaje e impacto| Analiza efectos en usuarios y operación                |                                                                              |
| Recopilación de rutas           | Ingreso de ruta                        | Registro inicial de recorridos                         | Registra y organiza la información operativa de las rutas.                  |

## Módulo 2: Gestión de Flota  
Este módulo asegura que tanto los vehículos como los conductores estén en condiciones operativas y legales. Se enfoca en el control físico y humano de los activos móviles.

### 2.1 Gestión de Vehículos

| Nivel 3 (Función)               | Nivel 4 (Sub-Función)                  | Nivel 5 (Detalle)                                      | Explicación                                                                 |
|---------------------------------|----------------------------------------|--------------------------------------------------------|------------------------------------------------------------------------------|
| Registro de vehículo            | Configuración                          | Ajustes iniciales del vehículo                         | Administra datos básicos de cada unidad.                                    |
|                                 | Registro de socios                     | Vinculación con propietarios o empresas                |                                                                              |
| Gestión de equipos              | Configuración                          | Parametrización de equipos                             | Controla dispositivos y sistemas asociados a la flota.                      |
|                                 | Procesamiento de datos                 | Integración por hardware                               | Conexión de sensores físicos (e.g., tacómetros, GPS).                       |
|                                 |                                        | Integración por software                               | Sincronización con sistemas digitales.                                      |
|                                 |                                        | Integración por servicios                              | Uso de plataformas externas (e.g., telemática avanzada).                    |
| Mantenimiento                   | Preventivo                             | Análisis de fallas                                     | Identificación temprana de problemas (mantenimiento predictivo).           |
|                                 |                                        | Programación de mantenimientos                         | Calendario de revisiones basado en tiempo/kilometraje.                      |
|                                 | Correctivo                             | Reparaciones tras incidentes o fallas inesperadas      | Garantiza la operatividad de la flota.                                      |
| Gestión de Recursos             | Control de inventario                  | Registro de repuestos y consumibles                    | Controla insumos y materiales.                                              |
|                                 | Optimización de recursos               | Uso eficiente de materiales                            | Ej. gestión de neumáticos.                                                  |
| Monitoreo en tiempo real        | Geolocalización                        | Seguimiento de vehículos en ruta                       | Supervisa ubicación y estado de la flota.                                   |
|                                 | Monitoreo de estado                    | Control de condiciones operativas                      | Ej. rendimiento del motor.                                                  |
| Inventario de repuestos         | Ingreso                                | Registro de nuevos repuestos al almacén                | Administra entradas y salidas de piezas.                                    |
|                                 | Salida                                 | Control de uso de piezas para mantenimiento            |                                                                              |
| Integración con proveedores externos | Ingesta                          | Recepción de datos externos                            | Ej. informes de talleres.                                                   |
|                                 | Preparación                            | Adaptación de datos recibidos para uso interno         |                                                                              |
|                                 | Intercambio                            | Enviar                                                 | Transmisión de datos (e.g., reportes de uso).                               |
|                                 |                                        | Consumir                                               | Uso de datos externos (e.g., precios de combustible).                       |

### 2.2 Gestión de Conductores

## Módulo 4: Información al Usuario  
Este módulo se centra en la comunicación bidireccional de información relevante al usuario (o cliente B2B) sobre el servicio.

### 4.1 Información de la flota

| Nivel 3 (Función)               | Nivel 4 (Sub-Función)                  | Nivel 5 (Detalle)                                      | Explicación                                                                 |
|---------------------------------|----------------------------------------|--------------------------------------------------------|------------------------------------------------------------------------------|
| Planificación de viajes         | GTFS                                   | Dinámicos                                              | Datos en tiempo real sobre la posición y retrasos de los buses.             |
|                                 |                                        | Estáticos                                              | Información fija sobre horarios y rutas planificadas.                       |
| Rutas                           | Horarios                               | Típico                                                 | Horario normal de operación.                                                |
|                                 |                                        | Fines de semana                                        | Horario modificado por menor demanda.                                       |
|                                 |                                        | Especial                                               | Horarios para festivos o eventos específicos.                               |
|                                 | Paradas                                | Información precisa sobre la ubicación de las paradas  |                                                                              |
| Vehículos                       | Geolocalización                        | Ubicación exacta del bus en el mapa en tiempo real     | Proporciona la ubicación y datos de movimiento del bus.                     |
| Integración con proveedores externos | Ingesta                          | Recepción de datos de plataformas externas             | Ej. clima, tráfico.                                                         |
|                                 | Preparación                            | Adaptación del formato de datos para el intercambio    |                                                                              |
|                                 | Intercambio                            | Enviar                                                 | Publicar datos del servicio a aplicaciones externas (e.g., Google Maps).    |
|                                 |                                        | Consumir                                               | Recibir y utilizar información de terceros (e.g., mapas de base).           |

### 4.2 Información de incidencias

|---------------------------------|----------------------------------------|--------------------------------------------------------|------------------------------------------------------------------------------|

|                                 |                                        | Vehículo                                               | Gestionar quejas sobre el estado o fallas del bus.
